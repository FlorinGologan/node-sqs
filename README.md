# node-sqs
Basic Node AWS SQS implementation:
* push messages - http://localhost:3003/
* read and delete messages - see console logs

# Install

``` 
npm i
```

# Run

```
npm start
```

Using VSCode `REST Client` run `test.http` file and see the logs from console.
